# 4. 포스팅하기

[여기서](https://jekyllrb.com/docs/posts) 포스팅 포맷을 참고합니다.

저는 아래와 같이 복사하여, 포스팅 했습니다!

```
---
layout: post
title:  "Welcome to Jekyll!"
---

# Welcome

**Hello world**, this is my first Jekyll blog post.

I hope you like it!
```
